```text
Vacancy Assignment Method
Setup
|
Pre-Interview Questionnaire

Manual

The user who has been assigned with a role to assign recruiters, shall manually assign the vacancies to respective recruiters.

Round robin

Based on the functional locations assigned to the recruiters, system will automatically assign created vacancies one after the other in a sequence irrespective of the vacancy position created.

Workload

In this method, system shall automatically assign the vacancies to the recruiters considering the number of available vacancies that each recruiter has been assigned with.
```
